## 2) Section Plan (Long-form One-Page)

- Sticky top navigation with anchor jump links
- Hero: premium value proposition + dual CTA + mockup cards
- Architecture: coach flow and client flow side-by-side
- Role sections: coach/admin tools, then client experience
- Feature block: workouts, nutrition, progress, notifications, subscriptions, analytics
- Dedicated meal import block: CSV template download + schema preview + import steps
- Comparison: manual coaching vs CubeFit
- Screen gallery: mapped cards for all core screens and modals
- Media demo section: video support and visual direction
- FAQ
- Final conversion CTA
- Footer with brand and trust placeholder

## 3) Visual Style Direction

- Positioning: luxury fitness technology, cinematic and precise
- Primary background: deep charcoal/black with subtle gradients
- Accent colors: electric green + cyan (limited, consistent usage)
- UI language: glassmorphism-inspired cards, soft depth, clean border glow
- Typography: premium sans-serif with bold, compact headlines
- Motion: subtle reveal-on-scroll, smooth hover lift, no playful effects
- Component style: rounded cards, controlled spacing, high-contrast readability

## 4) Content Outline & Messaging

- Core promise: "Train with structure. Coach with clarity. Progress with proof."
- Coach message: "Scale high-touch coaching without losing quality."
- Client message: "Follow clear plans with consistent accountability."
- Feature narrative:
  - Workouts: plans, days, exercise details, media, completion
  - Diets: plan composition, meals, macros, calories
  - Imports: bulk CSV meal import using official template schema
  - Progress: weight logs, weekly trends, coach visibility
  - Notifications: reminders and execution consistency
  - Subscription: trial/active/expired access logic
  - Analytics: decision-ready reporting and weekly snapshots
- CTA language:
  - Explore Product Screens
  - See How It Works
  - Join Early Access
  - Book Demo
  - Download Press Kit

## 6) Asset List & Placeholder Direction

### Must-have visual assets

- Hero banner composition
- Device mockups (coach dashboard + client app)
- Workout preview, diet preview, progress chart preview
- Import flow visual (template preview + upload confirmation state)
- Notification preview
- Branded icon set
- Logo marks (wordmark + icon)
- Gradient textures and abstract athletic backgrounds

### Placeholder art direction

- Tone: editorial fitness, premium, clean, focused
- Lighting: dramatic but controlled contrast
- Composition: modern devices angled in perspective, minimal UI clutter
- Avoid: generic stock gym poses, cartoon icons, noisy backgrounds

## 7) Reusability for Future Product Work

- Section copy is modular and can map directly into app onboarding
- Screen cards act as UI inventory and user journey reference
- Feature blocks can seed roadmap docs, app store copy, and investor decks
- Brand tokens in CSS provide a base for future design system expansion
- CSV import schema can be reused for in-app onboarding and support docs
